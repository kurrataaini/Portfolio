Drug Repurposing for COVID-19 using Text Mining and Clustering

In this project, we conducted a drug repurposing study for COVID-19 by applying text mining techniques to abstracts of scientific papers containing chemical-related information. We used the CORD-19 dataset to collect all abstracts, then performed a series of data cleaning steps, including removing blank and duplicate entries, and filtering out entries with missing abstracts by checking DOIs.

Next, we applied Drug Named Entity Recognition (DNER) using the PubTator API from NCBI to extract drug names from each abstract. Abstracts containing chemical entities were selected for further analysis.

We then proceeded with text preprocessing, which included case folding, removing punctuation, stopword removal, number removal, stemming, and transforming the text using TF-IDF vectorization.

For pattern discovery, we applied K-Means clustering to group similar abstracts. After clustering, we performed manual labeling into four classes and validated the results by comparing them with data from DrugBank. Finally, we used Principal Component Analysis (PCA) to visualize the clustering results.
